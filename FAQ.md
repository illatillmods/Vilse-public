# FAQ

## Is Vilse released?

No. Vilse is a private prototype and long-term project. There is no public build
or release date in this mirror.

## Is this repository open source?

No. This public mirror is documentation only. It is provided for public review,
product transparency, and project updates.

## Can I copy or reuse the idea, docs, or product direction?

No permission is granted to copy, modify, redistribute, host, resell, or use this
work or any substantial part of it in a competing product or service without
explicit written permission.

## What is actually implemented?

The private prototype demonstrates a dense Confluence Basin slice with
server-authoritative hauling, payout, records, markets, compliance, permits,
construction, labor, research, production, vehicle service, infrastructure, and
archive/heritage surfaces. These are prototype systems, not proof of a shipped
MMO.

## What is not implemented?

Production persistence, real MMO concurrency, public accounts, broad geography,
final art, and a public test build are not implemented here.

## Why not publish the code?

The private repository contains source, implementation details, internal tooling,
planning history, and product-specific logic that should not be public or
cloneable.

## What kind of game is this?

Vilse is a civilization MMO project focused on industry, infrastructure,
institutions, information access, logistics, permits, records, and long-term
consequence. It is not a blank survival reset or a theme-park quest funnel.
