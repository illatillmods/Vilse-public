# Public Assets

This folder is reserved for Vilse media that can be shown publicly without
leaking private implementation detail.

Allowed here:

- reviewed screenshots from public-safe builds
- concept images cleared for public use
- placeholders made specifically for the mirror
- diagrams that explain product direction without exposing source assets

Not allowed here:

- raw captures from private tools or debug builds
- internal filenames, logs, coordinates, IDs, overlays, or admin panels
- maps, source assets, shaders, data exports, or UI states that make the private
  prototype easier to clone
- customer, tester, account, telemetry, or private planning data

Until an asset has been reviewed, this folder should stay boring on purpose.
